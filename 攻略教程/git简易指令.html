git简易指令

创建git远程仓库
前置条件：正确配置git
在电脑上创建一个文件夹，右键运行bash的终端，出入指令
第一次创建远程仓库需要提交邮箱和用户名（或者直接连接github账号），按提示来即可




第一次创建指令
echo "# yqx.github.io" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https:***********（你仓库的地址）
git push -u origin main







日常推送更新文件的简答指令
git add .
git commit -m "web"
git push -u origin main
