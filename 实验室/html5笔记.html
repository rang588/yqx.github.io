图标 技术 描述

单标签类：

br break 换行(单标签)

文本样式类：

div divide 划分出一整行的空间
span 跨度宽度 划分出一小块空间
br break 换行(单标签)
p paragraph 段落
h1~6 head 标题等级
strong 文字加粗
em emphasis 倾斜强调
del delete 删除线
u underline 下划线

图片类：

src source 资源
URL： Uniform Resource Locator 文件路径
img基础: < img src="图像名.格式+URL" >在同一文件路径下不用写URL
title 图片提示内容 < img src="图像名.格式+URL" title="图片提示内容">
alt 图片显示不出来的时候的替换 < img src="图像名.格式+URL" alt="替换内容">
width height 设置长宽数值
border 设置边框粗细数值(html不太支持，一般通过css)

链接类：

a anchor 锚
href Hypertext Reference 超文本链接（URL填写#，则为空链接）
target 不填写默认为_self(覆盖原窗口) 填写_blank为打开新窗口
< a href="URL" target="类型">内容(文字，图片...) </a>
定位网页内部词条 herf中填入对应词条id图片类 然后对应标题标签中写入id < h2 id="图片类">图片类：</h2>

表格类：

table 表格
tr table row 行 th table heah 表头(相当于文字加粗)
td table data(表格数据)
thead 表头区域 必须有tr
tbody 表体

需要填入:

种类一:

实例： < table align="center" .....></table>
align 排列(对齐) left center right
border 默认无边框 有边框：1
cellpadding 文字和边框的距离 填入数值
cellspacing 边框和边框的距离 填入数值
width 填数
height 填数

种类二:

实例：< td colspan="跨越合并的数量"></td>
rowspan 跨越行合并
colspan 跨越列合并

表单类：

格式
< form action=" " method=" " name=" " >
//action:url服务器接收地址
//method:提交方式get或post
//name:表单名
联系方式： < input type=" " name=" " value=" " checked=" " maxlength=" ">
//表单按钮控件
//type:text长方框 password字符隐藏框 radio 单选勾选 checkbox 多选框
//type:submit 提交按钮 reset 清空表单所有数据 button 可以点击的按钮(和js一起) file 上传文件
//name 控件指定命名 防止提交信息混乱 单选框和多选框的name要相同
//value 仅仅对于部分可显示框内提示 其他value是为程序员后台看
//checked 填入checked对于单选和多选的默认选中
//maxlength text最多输入的字符数量
< input type=" " id="男生">
< label for="男生">锚点名< /label>
//通过label与控件的id建立联系 直接点击锚点名直接跳转到控件内部
//下拉列表格式
< select>
< option >安南 </option>
< option >北漠 </option>
< option >中土 </option>
< option select="select" >京都 </option>
//select="select" 默认的下拉列表选项 < /select>
//文本域格式(产生大的文本方框：一般用于写留言)
< textarea>
该框写留言
< /textarea>
